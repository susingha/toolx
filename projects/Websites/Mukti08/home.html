INTRO

Mukti is a National level annual technical symposium on Linux and related Free and Open Source Software(FOSS). It is a unique event of its kind among engineering colleges across India. There is something on offer for everyone, ranging from the SYSTEM ADMINISTRATION contest for the tech savvy to GAMING contests for pure fun.
 

Kon-figure


In this overnight event the participants have to accomplish a Server related task.

Code Cracker


An online programming contest where codes will be judged on real time basis.

Jigyasa


Challenge yourself in a brainstorming session of GENERAL QUIZ.

nCore


Core Departmental Events where Linux and FOSS reach beyond the boundaries of silicon branches.

Brain Mesh


Design and Implement a theme based WEBSITE to promote a company's e-presence.

Rush Hour


Fight one another out in an exciting and ADRENALINE pumping GAME to be played on Linux platform.
